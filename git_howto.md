# Подсказка по Git

Инициализация репозитория
```sh
git init
```

Добавить файл в папку
```sh
git add
```

Добавить коммит к действию
```sh
git commit -m
```

Вывести развернутый лог версий
```sh
git log
```

Вывести однострочный лог версий
```sh
git log --oneline
```

Перейти к какой-либо версии
```sh
git checkout
```

```sh
git branch
```
